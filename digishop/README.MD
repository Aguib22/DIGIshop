Ce projet est un exercice sur le quel je me suis entrainer.
il s'agit d'une application de vente en ligne (site Ecommerce)
les technologies utilisés sont:
    le html et css;
    le bibliothecque django de python;
    le javascript et sa bibliothecque jquery;

Comment puis-je executer cette application:
1-installer django la dernière version 4.1 :"python -m pip install -r django 4.1 /fichier";
2-créer une base de donnée en mysql: "create database nom_db" dans ce cas si remplacer nom_db par db_digishop;
3-effectuer les migrations en tapant les commandes suivantes:
    'python manage.py makemigrations'
    'python manage.py migrate'
4- lancer le serveur : 'python manage.py runserver'

apres cela pouvez ouvrir votre navigateur et tapez sur la bare de recherche "localhost:8000"
vous serez ensuite rediriger sur la page d'acceuil de l'application et là vous pouvez voir les different produits 
    vous pouver cliquez sur le bouton detail ou sur l'image pour voir les details du produit.
    appuiyer sur ajouter pour ajouter le produit dans votre panier.
    c'est tres simple l'utilisation de l'application jusqu'à la validation de la commande.

une vous valider votre commande l'administrateur de l'application recevra automatiquement votre requête.
